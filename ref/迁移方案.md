# 袋鼠君小程序 → 独立站后端 微信/谷歌登录迁移方案

## 一、概述

将袋鼠君小程序（PHP/ThinkPHP后端）的微信登录功能迁移到袋鼠君独立站后端（NestJS），同时后续新增谷歌登录。

### PHP 参考源码路径
- `ref/php-api/Login.php` — 微信登录控制器（weapp/wechat/bindwxphone/bindmobile/sendsms）
- `ref/php-api/UserModel.php` — 用户模型（wechatLogin/bindMobile）
- `ref/php-library/WechatApp.php` — 微信小程序SDK封装

### 独立站后端路径
- `~/workspace/kangaroo-japan-backend/` — NestJS 项目
- `src/auth/` — 现有的邮箱密码认证模块
- `src/users/` — 用户模块（User entity + UsersService）

### 需要新增的表（NestJS 迁移）

```sql
-- 第三方授权表（支持微信、谷歌、Apple等多种登录）
CREATE TABLE "user_providers" (
  "id" uuid NOT NULL DEFAULT uuid_generate_v4(),
  "user_id" uuid,                           -- 关联的用户ID（NULL表示未绑定手机号）
  "provider" character varying NOT NULL,    -- 提供商: 'wechat', 'google', 'apple'
  "provider_user_id" character varying NOT NULL,  -- 提供商的用户ID（微信openid / Google sub）
  "unionid" character varying,              -- 微信unionid（仅微信）
  "type" character varying,                 -- 微信小程序类型: 'weapp', 'gxsweapp', 'hwweapp'
  "name" character varying,                 -- 昵称
  "avatar_url" character varying,           -- 头像URL
  "session_code" character varying,         -- 微信session_code（用于checkweapp）
  "session_key_encrypted" character varying, -- 加密后的session_key
  "raw_metadata" jsonb,                     -- 原始用户信息JSON
  "created_at" TIMESTAMP NOT NULL DEFAULT now(),
  "updated_at" TIMESTAMP NOT NULL DEFAULT now(),
  CONSTRAINT "PK_user_providers" PRIMARY KEY ("id"),
  CONSTRAINT "FK_user_providers_user" FOREIGN KEY ("user_id") REFERENCES "users"("id") ON DELETE CASCADE,
  CONSTRAINT "UQ_user_providers_provider_userid" UNIQUE ("provider", "provider_user_id")
);

-- 短信验证码表
CREATE TABLE "sms_codes" (
  "id" uuid NOT NULL DEFAULT uuid_generate_v4(),
  "phone" character varying NOT NULL,
  "code" character varying NOT NULL,
  "type" character varying NOT NULL,        -- 'bind', 'login', 'verify', 'change'
  "expires_at" TIMESTAMP NOT NULL,
  "used" boolean NOT NULL DEFAULT false,
  "created_at" TIMESTAMP NOT NULL DEFAULT now(),
  CONSTRAINT "PK_sms_codes" PRIMARY KEY ("id")
);

-- 国家区号表（静态数据）
CREATE TABLE "country_codes" (
  "id" SERIAL PRIMARY KEY,
  "code" character varying NOT NULL,        -- 区号，如 "+86"
  "name" character varying NOT NULL,        -- 国家名
  "name_zh" character varying,              -- 中文名
  "country_code" character varying,         -- ISO 国家代码
  "is_show" boolean DEFAULT true
);
```

## 四、微信登录核心流程（NestJS 实现逻辑）

### 4.1 weapp 接口（微信小程序授权登录）

```
请求: POST /api/v1/wechat/weapp
Body: { code, encryptedData, iv, appid, type? }

流程:
1. 根据appid选择微信小程序配置
2. 调用微信code2session接口获取openid + session_key
3. 解密encryptedData获取用户信息（nickname, avatar, gender等）
4. 查找user_providers表:
   a. 已有记录且已绑用户(user_id != null) → 直接生成JWT登录
   b. 已有记录但未绑用户 → 生成临时JWT（含user_provider_id）
   c. 无记录 → 创建新记录 → 生成临时JWT（含user_provider_id）
5. 返回: { token, mobile: 0/1 }
   - mobile=1: 已绑定手机号，正常登录
   - mobile=0: 未绑定手机号，需要引导绑定
```

### 4.2 wechat 接口（微信公众号/H5登录）

```
请求: POST /api/v1/wechat/wechat
Body: { code, appid }

流程:
1. 根据appid选择微信配置
2. 调用code2session获取openid
3. 同weapp流程3-5，但无用户昵称头像
```

### 4.3 bindwxphone 接口（微信手机号绑定）

```
请求: POST /api/v1/wechat/bindwxphone
Body: { token, code, encryptedData, iv, appid }

流程:
1. 验证token（临时JWT，包含user_provider_id和user_key）
2. 调用code2session + decryptData获取手机号
3. 查找或创建用户
4. 绑定user_provider到user
5. 返回新JWT（正式登录token）
```

### 4.4 bindmobile 接口（短信验证码绑定手机号）

```
请求: POST /api/v1/wechat/bindmobile
Body: { token, mobile, code, inviteCode? }

流程:
1. 验证token（临时JWT）
2. 验证短信验证码
3. 查找或创建用户
4. 绑定user_provider到user
5. 返回新JWT（正式登录token）
```

## 五、JWT Token 设计

### 临时 Token（未绑定手机号）
```json
{
  "sub": "<user_provider_id>",
  "type": "provider_token",
  "user_key": "<md5(provider_user_id)>",
  "iat": ...,
  "exp": ...
}
```

### 正式 Token（已绑定手机号）
```json
{
  "sub": "<user_id>",
  "type": "user_token",
  "email": "...",
  "role": "user",
  "iat": ...,
  "exp": ...
}
```

## 九、关键差异点（PHP → NestJS）

| 项目 | PHP (ThinkPHP) | NestJS (TypeORM) |
|------|---------------|------------------|
| 主键 | 自增int | UUID |
| 用户表名 | `st_users` | `users` |
| 微信表 | `st_user_wechat` | `user_providers`（复用多平台） |
| Token类型 | 单一JWT（PHP JWT库） | NestJS JwtModule + Redis refresh token |
| 密码加密 | `sha1(md5(pass).salt)` | `bcrypt` |
| 微信SDK | 自定义WechatApp类 | 自定义WechatSdkService（nestjs + axios） |
| 数据库 | MySQL | PostgreSQL |
| ORM | ThinkPHP ORM | TypeORM |
